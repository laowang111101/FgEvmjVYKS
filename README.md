# 项目简介

## 核心功能
- 数据采集
- 预处理
- 特征提取
- 分类器设计

## 适用场景
- 农业领域
- 番茄植株病虫害自动识别与分类

- 结合迁移学习提高模型泛化能力
- 实现实时监测与智能反馈
- 辅助农业专家精准诊断
- 提升病虫害防治效率
